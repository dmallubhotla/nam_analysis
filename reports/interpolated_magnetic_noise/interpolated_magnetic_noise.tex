	\section{Interpolated dielectric constant} \label{sec:intro}
	Very similar to the electric noise, except trying to plot magnetic noise, again using interpolated Nam form\supercite{Nam1967}.
	Let's see what happens!
	In \fref{fig:cutoff} we can see how the cutoff parameter affects the magnetic noise.
